# 🏦 bank_scrapper

A set of Python-based web scrapers that automate the downloading of circulars, notices, and regulatory disclosures from the websites of major Indian banks and mutual fund providers.

---

## 📁 Contents

| Script Name              | Description |
|--------------------------|-------------|
| `axis_bank_scrapper.py`  | Scrapes public disclosures or circulars from Axis Bank's regulatory page. |
| `hdfc_bank_scrapper.py`  | Extracts circulars, notices, or scheme documents from HDFC Mutual Fund or Bank's disclosures. |
| `kotak_bank_scrapper.py` | Scrapes regulatory updates and notices from Kotak Mutual Fund's official site. |
| `nippon_bank_scrapper.py`| Downloads scheme documents or announcements from Nippon India Mutual Fund's website. |
