# 🏛️ gov_scrapper

A collection of Python-based scrapers designed to automate the extraction of official circulars, regulations, and notices from key Indian government and regulatory bodies:

- **IRDAI (Insurance Regulatory and Development Authority of India)**
- **SEBI (Securities and Exchange Board of India)**
- **MCA (Ministry of Corporate Affairs)**
- **NPCI (National Payments Corporation of India)**

---

## 📄 Scripts Overview

| Script Name                  | Description |
|-----------------------------|-------------|
| `egazette_irdai_scrapper.py`| Scrapes IRDAI circulars or e-gazette content. |
| `egazette_sebi_scrapper.py` | Extracts SEBI circulars or notifications published via e-gazette. |
| `mca_scrapper.py`           | Downloads and organizes circulars from the MCA portal, with folder-based segregation. |
